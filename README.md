# Smart IoT Door Lock with Emergency Alert System

## Overview
This project proposes a Smart IoT Door Lock system designed to improve home security by monitoring door access events and providing emergency alert capabilities.

The system records lock and unlock activities, detects unauthorized access attempts, and allows users to trigger emergency alerts through a portable remote button.

## Key Features
- Real-time door lock monitoring
- Activity logging with timestamps
- Unauthorized access detection
- Emergency alert button
- Remote monitoring through web or mobile interface

## System Components
- Smart Door Lock Sensors
- Microcontroller (ESP32 / Arduino)
- Wi-Fi Communication
- Backend Server
- Web / Mobile Monitoring Interface

## System Workflow
1. Door lock interaction detected.
2. Sensors send event data to the microcontroller.
3. Microcontroller processes the event.
4. Event data is sent to the backend server.
5. Activity logs are stored with timestamps.
6. Alerts are generated for unauthorized access or emergency triggers.
